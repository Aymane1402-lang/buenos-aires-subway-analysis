# Buenos Aires Subway Analysis 🚇

Analysis of passenger traffic in the Buenos Aires subway 
system using Python, Pandas and Matplotlib.

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Findings
- Peak hours: 16:00 - 18:00
- Busiest line: Line E
- Busiest stations: Bolívar, Retiro E, Correo Central
- 44.6% of busiest records occur during afternoon rush hour

## Dataset
March 2024 - Buenos Aires subway passenger data
